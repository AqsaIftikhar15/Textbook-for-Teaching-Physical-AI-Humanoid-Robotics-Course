# ROS 2 Fundamentals

## Learning Objectives

- Understand ROS 2 middleware architecture
- Explain Nodes, Topics, Services, and Actions conceptually
- Understand message passing and communication patterns in ROS 2
- Read URDF specifications for humanoid robots

## Overview

ROS 2 (Robot Operating System 2) provides the middleware layer that enables communication between different components of a robotic system. It serves as the "nervous system" of the robot, coordinating sensors, actuators, and computational processes.
