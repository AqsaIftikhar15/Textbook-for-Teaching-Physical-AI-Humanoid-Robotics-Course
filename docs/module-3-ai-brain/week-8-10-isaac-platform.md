# NVIDIA Isaac Platform

## Learning Objectives

- Understand NVIDIA Isaac SDK and Isaac Sim setup and configuration
- Learn about photorealistic simulation and synthetic data generation
- Understand AI-powered perception and manipulation pipelines
- Learn about reinforcement learning for robot control
- Understand sim-to-real transfer techniques

## Overview

The NVIDIA Isaac platform provides tools for developing AI-powered robotic applications, with particular focus on perception, navigation, and manipulation capabilities for humanoid robots.
