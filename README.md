# Physical AI & Humanoid Robotics Book

This repository contains the educational book on Physical AI & Humanoid Robotics, structured as four progressive modules:

1. **Module 1: The Robotic Nervous System (ROS 2)** - 4 weeks covering ROS 2 fundamentals
2. **Module 2: The Digital Twin (Gazebo & Unity)** - 2 weeks covering physics simulation
3. **Module 3: The AI-Robot Brain (NVIDIA Isaac)** - 5 weeks covering AI perception and control
4. **Module 4: Vision-Language-Action (VLA)** - 1 week covering multimodal integration

## Setup

To run this documentation site locally:

1. Ensure you have Node.js >= 18.0 installed
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
